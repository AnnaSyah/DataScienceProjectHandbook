## 📌 Overview

This repository contains hands-on case studies from *Chapter 2: Python for Data Science* in the **Data Science Project Handbook**.

The project is designed to help learners understand how Python concepts can be applied to practical Data Science scenarios using clean, beginner-friendly, and practical implementations.

> 🌏 Current implementation is primarily optimized for Indonesian-language academic documents and interactions.

---

## 🔍 Case Studies

### Case Study 2.1 – Sentiment Analysis of Customer Reviews

Analyze customer reviews using Python comprehensions and sentiment scoring techniques.

### Case Study 2.2 – Analyzing Customer Purchase Patterns in E-Commerce

Explore customer shopping behavior using `lambda`, `map`, `filter`, and `reduce` functions.

### Case Study 2.3 – Deep Dive into Customer Purchase Patterns

Perform deeper analysis of transaction data using functional programming approaches in Python.
